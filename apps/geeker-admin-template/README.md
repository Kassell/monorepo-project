# Geeker-Admin

### 介绍 📖

> 这是一个极简的 Geeker-Admin 管理后台。它只包含了 ElementPlus & axios & iconfont & permission control & lint && cz-git && 主题布局切换，这些搭建后台必要的东西。

### 安装使用步骤 📔

- **Clone：**

```text
# GitHub
git clone https://github.com/HalseySpicy/Geeker-Admin/tree/template
```

- **Install：**

```text
pnpm install
```

- **Run：**

```text
pnpm dev
pnpm serve
```

- **Build：**

```text
# 开发环境
pnpm build:dev

# 测试环境
pnpm build:test

# 生产环境
pnpm build:pro
```

- **Lint：**

```text
# eslint 检测代码
pnpm lint:eslint

# prettier 格式化代码
pnpm lint:prettier

# stylelint 格式化样式
pnpm lint:stylelint
```

- **commit：**

```text
# 提交代码（提交前会自动执行 lint:lint-staged 命令）
pnpm commit
```
